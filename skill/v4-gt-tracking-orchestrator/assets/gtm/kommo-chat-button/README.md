# Kommo Website Chat Button GTM Template

User-provided community template asset for injecting the Kommo website chat button script.

## Important

- Enter only the installation key generated in the target client's Kommo account.
- Do not reuse IDs, hashes or keys from another client.
- This template does not capture UTMs, click IDs, first/last touch or CRM conversion events.
- Test in GTM Preview, document permissions, obtain approval and publish with a rollback version.
- Retain the bundled license.
